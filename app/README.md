## Stack

- Typescript
- Vue (Component-driven development and reactivity)
- Vite (Bundling)
- Pinia (Reactive data store)
- Axios (Http requests)

## Testing

- Vitest (Unit testing)
- Storybook (Isolated component development)
- msw (Mocking http requests)
- Cypress (E2E testing)

## Styling

- SCSS
- Base CSS resets and themes
- Bulitin functional CSS library:
  - Utilities classes (flex:rows-lg, m-auto, w-fit)
  - Visual patterns classes
